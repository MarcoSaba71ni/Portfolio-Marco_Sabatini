# Portfolio-Marco_Sabatini
Marco Sabatini Portfolio

## Scripts

- `npm run dev`: start Tailwind in watch mode while developing
- `npm run watch`: same as dev, watches `src/input.css` and writes `css/style.css`
- `npm run build`: generate minified production CSS

## Deploy Checklist

1. Install dependencies: `npm install`
2. Build production CSS: `npm run build`
3. Commit and push updated static files (`index.html`, `css/style.css`, `js/*`, assets)
4. Deploy to your static host (GitHub Pages, Netlify, Vercel static output, etc.)
